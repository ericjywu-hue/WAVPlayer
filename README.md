# ♫ WAV 音效播放器 (WAV Sound Player)

一個以 C# Windows Forms 開發的 WAV 音效檔播放器，支援播放、重複播放、停止等功能，並具備深色主題介面與多項貼心輔助設計。

---

## 📋 專案簡介

本專案為「視窗程式設計 (II)」課堂練習作業，使用 `System.Media.SoundPlayer` 類別播放 WAV 格式音效檔，並在原始功能基礎上加入多項創意擴充。

### 主要功能

- **播放一次**：載入並播放指定的 WAV 音效檔（`F5`）
- **重複播放**：循環播放指定的 WAV 音效檔（`F6`）
- **停止播放**：停止目前正在播放的音效（`F7` / `ESC`）
- **瀏覽檔案**：開啟檔案對話方塊選擇 WAV 檔（`Ctrl+O`）
- **最近使用清單**：下拉選單快速重選曾使用過的檔案
- **拖曳放置**：直接將 `.wav` 檔案拖曳至視窗即可載入
- **檔案資訊顯示**：載入後顯示檔名、檔案大小與修改日期
- **狀態列回饋**：播放狀態即時顯示於介面底部，無多餘彈跳視窗
- **跑馬燈效果**：播放中以動態文字顯示目前播放的檔名

## 🚀 執行說明

### 環境需求

| 項目 | 需求 |
|------|------|
| 作業系統 | Windows 10 / 11 |
| 開發工具 | Visual Studio 2019 或以上 |
| 目標框架 | .NET Framework 4.7.2 |
| 語言版本 | C# 7.3 |

### 執行步驟

1. 複製或下載本專案
2. 以 Visual Studio 開啟 `WAVPlayer.sln`
3. 按下 `F5` 或點選「開始」執行

### 使用方式

1. 點選 **📂 瀏覽** 按鈕，或按 `Ctrl+O`，選擇 `.wav` 音效檔
2. 也可以直接將 `.wav` 檔案**拖曳**到視窗中載入
3. 選擇播放方式：
   - **▶ 播放一次**（`F5`）：播放一次後自動停止
   - **🔁 重複播放**（`F6`）：持續循環播放
   - **⏹ 停止播放**（`F7` / `ESC`）：停止目前播放

## ✨ 創意擴充說明

| 功能 | 說明 |
|------|------|
| 深色主題 | 深藍/紫色系配色，搭配綠色播放、紫色重複、紅色停止按鈕 |
| 鍵盤快捷鍵 | `F5` / `F6` / `F7` / `ESC` / `Ctrl+O`，可完全無滑鼠操作 |
| 拖曳放置 | 直接將 `.wav` 拖進視窗即可載入，無須點按瀏覽 |
| 最近使用清單 | 下拉選單保留曾載入的檔案，快速重選 |
| 即時狀態回饋 | 狀態文字直接顯示於介面，以顏色區分就緒/播放/警告/錯誤 |
| 跑馬燈效果 | 播放中動態滾動顯示檔名，視覺活潑 |
| 檔案資訊 | 載入時顯示檔名、大小、最後修改日期 |
| 播放器統一管理 | 追蹤 `activePlayer`，切換播放模式時自動停止舊的，避免聲音疊加 |
| 關閉預設選「否」 | 關閉確認視窗預設按鈕為「否」，防止誤關 |

## 📌 注意事項

- 僅支援 `.wav` 格式音效檔
- `SoundPlayer` 為同步載入，建議使用較小的 WAV 檔以避免短暫延遲
